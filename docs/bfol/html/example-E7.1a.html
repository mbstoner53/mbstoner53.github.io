Theorem example-E7.1a 1080
Description: '𝑥' is conditionally not free in the statement '(𝑥 ⋅ 𝑦) = 0' when '𝑦 = 0'. †

Since no arithmetic axioms have been introduced, we state '(𝑥 ⋅ 0) = 0' as a hypothesis.
