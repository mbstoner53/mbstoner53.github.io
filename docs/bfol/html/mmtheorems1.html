3.2  Logical Equivalence
 
3.2.1  Motivation Theorems.

If we use 'equiv(𝜑, 𝜓)' as an abbreviation for '¬ ((𝜑 → 𝜓) → ¬ (𝜓 → 𝜑))', then it follows from lemmas mbifwd-P2.1a 100, mbirev-P2.1b 101, and mbicmb-P2.1c 102 that...

⊢ equiv(𝜑, 𝜓) ⇔ ⊢ (𝜑 → 𝜓) & ⊢ (𝜓 → 𝜑).

This proves that the formula 'equiv(𝜑, 𝜓) ≔ ¬ ((𝜑 → 𝜓) → ¬ (𝜓 → 𝜑))' precisely represents the property of logical equivalence between '𝜑' and '𝜓'. We can use this to our advantage in order to define '(𝜑 ↔ 𝜓)'.

 
Theoremmbifwd-P2.1a 100 Motivation for ↔ Definition, Part A.
⊢ ¬ ((𝜑 → 𝜓) → ¬ (𝜓 → 𝜑))    ⇒   ⊢ (𝜑 → 𝜓)
