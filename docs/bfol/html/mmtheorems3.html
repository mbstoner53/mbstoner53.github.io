4.2.2  Utility Recipes.

For the rest of the chapter we will treat P3.1 - P3.20 as a new set of axioms. From here on out, we will not rely on any results from chapters 1 or 2 (except indirectly through the new set of axioms).

This will allow us to track the use of ndexclmid-P3.16 191. By avoiding the Law of Excluded Middle, it is possible to develop an intuitionist logical framework alongside the standard classical case.

From here on, all theorems derived without reference to the Law of Excluded Middle (ndexclmid-P3.16 191) will have a † in the comment.

Theoremimsubd-P3.28c 281 Implication Substitution (dual) †.

This can also be called "Linking Syllogism" because the result is the statement that adding the missing inner link between the two hypotheses links the entire chain as a consequence.

The mneumonic is as follows...

If ( 1 → 2 ) and ( 3 → 4 ) then ( ( 2 → 3 ) → ( 1 → 4 ) )...

or ( left-middle → right-middle ) → ( left-outer → right-outer ).

One can substitute ( 1 → 1 ) for ( 1 → 2 ), or ( 3 → 3 ) for ( 3 → 4 ) to see how imsubr-P3.28b 279 and imsubl-P3.28a 277 are related as special cases.

⊢ (𝛾 → (𝜑 → 𝜓))    &   ⊢ (𝛾 → (𝜒 → 𝜗))    ⇒   ⊢ (𝛾 → ((𝜓 → 𝜒) → (𝜑 → 𝜗)))
