Theorem ndenfrand-P7.4 835
Description: Natural Deduction: Effective Non-Freeness Rule 4.

If '𝑥' is (conditionally) effectively not free in both '𝜑' and '𝜓', then '𝑥' is (conditionally) effectively not free in '(𝜑 ∧ 𝜓)'.
