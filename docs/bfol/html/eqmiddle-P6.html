Theorem eqmiddle-P6 714
Description: Introduce Intermediate Middle Variable from Equality.

'𝑦' cannot occur in either '𝛾' or '𝑡'.

Hypothesis
Ref Expression
eqmiddle-P6.1 ⊢ (𝛾 → 𝑥 = 𝑡)
Assertion
Ref Expression
eqmiddle-P6 ⊢ (𝛾 → ∃𝑦(𝑥 = 𝑦 ∧ 𝑦 = 𝑡))
