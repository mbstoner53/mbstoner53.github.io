Theorem ndbii-P3.13 188
Description: Natural Deduction: '↔' Introduction Rule.

If we have previously deduced both directions of a biconditional, then we can deduce the full biconditional.

ndbii-P3.13.1 ⊢ (𝛾 → (𝜑 → 𝜓))

Ref Expression
ndbii-P3.13 ⊢ (𝛾 → (𝜑 ↔ 𝜓))
