Theorem psubenfr-P6 790
Description: Proper Substitution Applied to ENF Variable.

If '𝑥' is effectively not free in '𝜑', then replacing '𝑥' with some '𝑡' through proper substitution has no effect on '𝜑'.

Hypothesis
Ref Expression
psubenfr-P6.1 ⊢ Ⅎ𝑥𝜑
Assertion
Ref Expression
psubenfr-P6 ⊢ ([𝑡 / 𝑥]𝜑 ↔ 𝜑)
